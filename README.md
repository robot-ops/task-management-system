# Task Management System

Aplikasi **Task Management System** sederhana yang dibangun untuk memenuhi kriteria *Technical Test: Fullstack Web Developer*. Proyek ini mencakup sistem autentikasi pengguna berbasis JWT serta operasi CRUD dan pemfilteran tugas pribadi.

---

## 📂 Panduan Dokumentasi Lengkap

Untuk mempermudah peninjauan dan instalasi proyek, dokumentasi teknis telah dipecah secara terstruktur ke dalam folder `docs/`:

1. 💻 [**Dokumentasi Backend (Setup & DB)**](docs/backend.md)
   * Prasyarat sistem (Node.js & MySQL).
   * Konfigurasi berkas `.env` dan variabel lingkungan.
   * Skema database (`users_table` & `tasks_table`).
   * Panduan instalasi dependensi dan menjalankan server Express.
2. 🎨 [**Dokumentasi Frontend (React.js)**](docs/frontend.md)
   * Setup proyek React dengan Vite.
   * Integrasi Bootstrap (melalui CDN) dan Axios client.
   * Alur state manajemen dan interaksi antarmuka pengguna (*user interface*).
3. 🔑 [**Dokumentasi API & Postman**](docs/api.md)
   * Daftar lengkap endpoint API (Auth & Task CRUD).
   * Format parameter *Request Body* JSON dan struktur *Response*.
   * Petunjuk penggunaan Postman Collection yang terintegrasi di folder [`postman/`].
